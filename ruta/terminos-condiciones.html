1. ¿Cuál es la información que recopilamos?

  • Sea de su conocimiento que los datos que son tomados de su navegador son los siguientes:
    • Hora: Esta es proporcionada por su navegador a través de una función y es tratada de manera correcta. Sin ningún uso maligno y es almacenada y usada de forma correcta.
    • Fecha: Esta es proporcionada por su navegador a través de una función y es tratada de manera correcta. Sin ningún uso maligno y es almacenada y usada de forma correcta.
    • Nickname: Este 'apodo' es proporcionado por el usuario final, el cual se recomienda que no sea ningún de los siguientes:
      • Nombres reales.
      • Nombres de personas de las cuales no se cuente con el derecho de uso, para poder regístralo a través de este y otros sitios web.
      • Así como el uso de nombres de marcas nacionales e internacionales, ya que dichas marcas pueden no aprobar el uso indebido de su marca y/o nombre jurídico.
    • Es por ello que hacemos hincapié en que el nickname sea creado manera ficticia, que no pueda perjudicar a ninguna persona, así como pudiera perjudicar a marcas personales o empresas del sector público y privado.

2. ¿Cómo usamos esta información?

  • Sea de su conocimiento cuales serán los usos que tendrán los datos, a los cuales tendremos acceso.
    • Hora: La hora que nos proporciona su navegador será usada para tener un control de las acciones que se pueden realizar dentro de este sitio web, como lo son:
      • Botón de likes: La hora será ocupada para saber la hora del día en que el usuario presiono este botón, para después restringirle el uso nuevamente de este botón
      • Comentarios: La hora será ocupada para saber la hora del día en la que el usuario hizo su comentario, para después poderle hacer un resumen de su actividad, como lo es: la hora en la que se efectuó el comentario.
    • Fecha: La fecha que nos proporciona su navegador será usada para tener un control de las acciones que se pueden realizar dentro de este sitio web, como lo son:
      • Botón de likes: La fecha será ocupada para saber la fecha en la que el usuario presiono este botón, para después restringirle el uso nuevamente de este botón
      • Comentarios: La fecha será ocupada para saber la hora del día en la que el usuario hizo su comentario, para después poderle hacer un resumen de su actividad, como lo es: la fecha en la que se efectuó el comentario.
    • Nickname: El nickname que nos proporciona el usuario es de único uso para poder identificar en la base de datos los diferentes usuarios que se registran en la base de datos. Así como tiene la función de servir como un identificador de usuarios.

4. ¿Qué pasa si no quiero compartir mi información?

  • Es comprensible que no quiera compartir sus datos con páginas que no son de mucha confianza, es por ello que le aconsejamos que si no tiene la disposición de compartir sus datos con este sitio web no lo haga, ya que está en todo su derecho de no hacerlo, o en dado caso de que quisiera visual el contenido puede hacerlo proporcionando datos que sean incorrectos a los reales.
